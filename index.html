Projects

Course: Capstone Project

Objective: Our objectives is to create a fun, educational application for investing in the stock market. Stock Overflow will have two main features: an educational mode where users can practice investing in stocks with real-time prices and a competitive fantasy stock trading league where users can compete against their friends to see who can make the highest gain over a set period of time. All of the money and purchases are fictional of course, but will be simulating real-life investing as close as is possible with our API limitations.

Technologies: This is a four person project that we'll be developing over the course of thirteen scrum sprints. We will manage our sprints with Confluence and JIRA, and utilize a Git repository hosted in BitBucket. The application will be written in Java using both the Spring and Hibernate frameworks. The frontend will be built with JavaServlet Pages (JSP), and data will be stored in a MySQL database.

Course: Natural Science Summer Research Program

Objective/Outcome: I worked on this research project with one other student and a professor from Benedictine. The focus of the project was to create a low-cost autonomous drone capable of detecting a reference object for search and rescue applications. By the end of the summer, the contribution of the work was a script capable of detecting a 3-dimensional reference object through post-processing of drone footage.

Technologies: Image processing was done with a Matlab script that detected, compared, and matched SURF in a cycle of images of a reference object and a single video frame. RANSAC and proportion comparisons were used to locate the reference object in the video frame and prevent misidentifying other objects. We piloted the drone with libraries written in either node.js or java.
